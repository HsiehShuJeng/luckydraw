# LuckyDraw

這是一個方便的免費線上抽獎程式，無論您是行銷抽獎活動、公司年度尾牙抽獎或是節慶活動抽獎，都能符合您的需求。

在未經同意時，請勿複製改寫用於公司商業用途。

[Lucky Draw](https://apan1121.github.io/luckydraw/)

[Demo Video](https://www.youtube.com/watch?v=Vp7fli021d8)
